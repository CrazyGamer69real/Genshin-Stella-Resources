--[ Genshin Stella Mod - Community presets ]--
This folder contains community-made ReShade presets for Genshin Impact, a popular open-world action RPG.


--[ Pull Requests ]--
If you want to add your own preset, just create a new Pull Request with screenshots. But hide your UID.

» Use valid filenames:
1. <Discord username> - <Small description>.txt
2. <GitHub username> - <Small description>.txt

» Examples:
1. Sefinek - HDR
2. sefinek24 - HDR

Repository    : https://github.com/sefinek24/Genshin-Stella-Resources
Pull Requests : https://github.com/sefinek24/Genshin-Stella-Resources/pulls


--[ Have fun! ]--
